# Бизнес-контекст проекта

## Название проекта

**QuadroTech**

## О чём проект

Внутренняя бизнес-папка с контекстом для автоматизации и внедрения ИИ-инструментов в действующую компанию по продаже компьютерной и печатной техники (Pantum, Skyros, ПК, оргтехника) через B2G-аукционы, B2B-проекты и маркетплейсы.

## Правило работы с контекстом

**При любой задаче сначала читай этот файл, затем открывай только те файлы, которые нужны именно для текущей задачи.** Не загружай все файлы сразу — в совокупности они большие, а почти любая задача требует лишь 2–3 конкретных.
