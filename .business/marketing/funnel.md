# Воронка продаж

## Воронка аукционов (B2G) — оценка 2025, на глаз

Точного учёта нет, цифры со слов Александра и по ощущениям:

- смотрим **~50 аукционов в неделю** (фильтрация по критериям)
- участвуем в **5–10 в неделю**
- выигрываем **1–2 в неделю**
- годовой итог — **~50–100 выигранных контрактов**

Всё это делает ~0.5 менеджера (один менеджер, но частью времени — см. [[team]]). Именно этот процесс — главный кандидат на автоматизацию в 2026 (см. [[annual]] и [[quarterly]]).

**Целевая задача автоматизации:**

- убрать человеческое ограничение на число просмотренных аукционов в неделю
- смотреть **не только Поволжье, а всю РФ**
- поднять конверсию на каждом шаге за счёт автоматического подбора оборудования и подготовки заявок
- дать возможность масштабировать воронку без линейного роста штата

## Воронка по B2B

Точных цифр нет. Логика сценариев:

**Повторный клиент / сарафан:**
входящий запрос → предложение / конфигурация под задачу → согласование → сделка → повторная покупка.

**Через тендер:**
участие в аукционе → победа → поставка без срывов → клиент потом выходит напрямую → повторные продажи минуя процедуру.

Учёт конверсий пока не ведётся — точка роста.

## Воронка по маркетплейсам

Логика стандартная для площадок:
показ карточки → клик → заказ → доставка → отзыв / повтор.

Сквозные метрики есть в отчётах Ozon, но нет единого дашборда с воронкой по всем каналам сразу (целевое состояние KPI — [[kpi]]).
