# Путь клиента

## Путь клиента по исторической модели

### Сегмент 1. Госзакупка

Появление спроса → подготовка ТЗ / публикация закупки → участие компании в процедуре → победа / контракт → поставка → исполнение без штрафов → повторные закупки / укрепление репутации.

### Сегмент 2. B2B (оргтехника, IT, ПК под заказ)

Два основных источника входа:

**Вход 1 — повторный клиент / сарафан:**
входящий запрос → обсуждение наличия и цены / конфигурация под задачу → согласование условий → поставка → повторные покупки.

**Вход 2 — через тендер:**
участие в аукционе → победа → поставка без срывов → клиент выходит повторно уже напрямую, минуя процедуру → переход в первый сценарий. Цифры воронки — [[funnel]].

Активного outbound нет — холодных продаж не ведём.

### Сегмент 3. Маркетплейсы

Поиск по артикулу / категории → сравнение цены и условий → заказ → доставка → отзыв / повторная покупка.

## Удержание

Сильнее всего удержание строилось не через бренд-маркетинг, а через:

- надёжность
- наличие
- цену
- знакомство с процессом закупки клиента
- репутацию поставщика
