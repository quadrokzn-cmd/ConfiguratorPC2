# КВАДРО-ТЕХ: сервис-конфигуратор ПК

Сервис подбирает комплектующие под запрос пользователя, проверяет совместимость и показывает минимальную цену по поставщикам.

## Стек
- Python + FastAPI
- PostgreSQL
- OpenAI API
- HTML/JS фронтенд (позже)

## Структура проекта
```
app/              — исходный код бэкенда
  main.py         — точка входа FastAPI
  config.py       — настройки
  database.py     — подключение к PostgreSQL
  models/         — модели SQLAlchemy
  schemas/        — схемы Pydantic
  routers/        — эндпоинты
  services/       — бизнес-логика
migrations/       — SQL-скрипты миграций
static/           — фронтенд (позже)
tests/            — тесты
```

## Как запустить
1. Создать БД PostgreSQL и применить `migrations/001_init.sql`.
2. Скопировать `.env.example` в `.env`, подставить свои значения.
3. `pip install -r requirements.txt`
4. `uvicorn app.main:app --reload`
